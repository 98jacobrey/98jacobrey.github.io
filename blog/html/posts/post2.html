PHP Auto Blog Posts

Automating blog posts with PHP GET and EXPLODE.

Posted on February 13, 2021

Ever wonder how to naturally fill a website with unique web pages that are directly tied to a database. Well, I did. After thinking about it and searching the web for about fifteen minutes I found that the answer is quite simple. Step one, create a template that you can use to fill with a database entry. Step two, populate a database with content that will be used to create your pages. Step three, create a way to identify what data to pull.

PHP has the capability to create a dynamic and automated website easily. Using tools like explode to gather post ids for the database and parse its content through a PHP template. This allowed me to make one template for all my blog posts that would then be populated by my PHP code, saving space and time.

PHP Fetch Code

$query = explode('/', parse_url($_SERVER['REQUEST_URI'], PHP_URL_PATH));
$qs = array(); foreach ($query as $key=>$value) $qs[$key] = urldecode(htmlspecialchars($value));
array_shift($qs);

Here we separate each section of the URL by each “/” to fill an array that will be used to later identify what to pull from our database. We then use the array to determine which table we use, and identify the primary key. After that we simply echo the appropriate content from the entry and then you have your blog post.
